# actions_build_ErfanGSIs #

## 这是一个基于Github Actions制作的自动跑ErfanGSI的脚本。 ##
## 步骤如下： ##
### 首先，fork这个仓库。 ###
### 接着，编辑build_ErfanGSIs.yml文件，把其中的ROM_URL改成你要做gsi的底包（注意直链）；还有ROM_NAME改成你的ROM名称。 ###
### 然后，按Star小星星就可以开始了！ ###
### 最后，可以在Actions里看到最终的GSI了！ ###
